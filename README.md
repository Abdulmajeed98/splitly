# Splitly

A group expense splitter — track who paid for what, calculate who owes whom, settle up.

This is a learning project. The stack mirrors `fib-backend-typescript`: NestJS + Drizzle + PostgreSQL + Effect/Either + pragmatic hexagonal architecture.
